Math

Source code

The math module can be imported with the statement import math.

The math module is used to do math. To do statistics, use the statistics module.

Constants

  • math.pi: Pi.
  • math.tau: Tau.
  • math.sqrt_pi: square root of pi (math.sqrt(math.pi)).
  • math.sqrt_tau: square root of tau (math.sqrt(math.tau)).
  • math.e: Euler's number.
