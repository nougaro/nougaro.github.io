Hello, world!

This is the “Hello, world!” program in Nougaro.

print("Hello, world!")

Alternatively:

import hello

If you want to use functionnal programming:

def main()
    print("Hello, world!")
end

main()

If you want to use object-oriented programming:

class Main
    def print_hello_world()
        print("Hello, world!")
    end
end

Main().print_hello_world()
